Revenue & Billing KPI Dashboard (Excel)

Overview
This project is an Excel dashboard that tracks revenue and billing performance across time, service categories, and regions. It summarizes billed vs. paid amounts, outstanding balances, exceptions, and aging buckets using PivotTables, PivotCharts, and interactive slicers.

Tools Used

Microsoft Excel (PivotTables, PivotCharts, GETPIVOTDATA, slicers)

Data cleaning + calculated fields (Days to Pay, Days Past Due, Aging Bucket, Outstanding Balance)

Key KPIs

Total Billed, Total Paid, Outstanding Balance

Collection Rate, Avg Days to Pay

Unpaid Invoice Count, $ Past Due 30+

Exception Rate

Dashboard Features

Monthly Revenue Trend (Billed vs Paid)

Outstanding Balance by Aging Bucket

Exceptions by Reason

Billed vs Paid by Service Category

Revenue Summary by Region

Slicers: Region, Service Category, Payment Status, Denial/Exception

How to Use

Open the Excel file

Use slicers on the right to filter the dashboard

KPIs and charts update automatically based on selections
